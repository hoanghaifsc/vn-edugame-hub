# Vietnam EduGame Hub

Mini game giáo dục cho học sinh tiểu học Việt Nam.

## Games hiện có

- **Vietnamese Word Quiz** — luyện từ vựng, chính tả, nghĩa từ (Grade 1–2–3)

## Cách chạy local

Mở `games/index.html` trực tiếp trên browser.

## Phong cách

Thiết kế theo vibe [vn-edugame-hub.vercel.app](https://vn-edugame-hub.vercel.app):
- Card-based, gọn, dễ dùng
- Phân loại theo môn / lớp
- Feedback ngay, không onboarding dài
